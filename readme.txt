W4 Cloudinary plugin is an extension of Cloudinary plugin. It adjusts the way images are being optimized for responsive websites.

== Description ==

== Installation ==

1. Install and activate the Cloudinary plugin by Cloudinary, ltd. - https://wordpress.org/plugins/cloudinary-image-management-and-manipulation-in-the-cloud-cdn
2. Upload the W4 Cloudinary plugin files to the `/wp-content/plugins/w4cloudinary` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the W4 Cloudinary plugin through the 'Plugins' screen in WordPress
4. Set your Cloudinary Cloud name in W4 Cloudinary settings
